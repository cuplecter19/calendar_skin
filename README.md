### calendar_skin by ...

- 아보카도 퍼스널, php 5.6 환경 기준으로 작성된 캘린더형 게시판 스킨입니다.
- 기능은 다음과 같습니다.

1. 월별 일정 추가/수정/삭제/복사/위젯 등록 여부를 지정 가능
2. 일정별로 색상 지정 가능 및 과거에 사용한 색상 히스토리 O
3. 캘린더 상에서 일정이 있는 날짜에 마우스 오버하면 툴팁으로 제목 표시
4. 일정 등록 시 반복 설정 (일간, 주간, 월간 및 반복 횟수를 숫자로 설정 가능)
5. 일정 추가/수정 모달에서 D-day 및 Goal(데드라인) 설정 - Goal의 경우 남은 일수를 세고, 당일이 되면 자동으로 사라지는 D-day의 변주격 기능입니다. 마감일 등의 설정에 이용하세요.
   등록한 D-day와 Goal은 캘린더의 'D-day' 버튼을 누르면 모아서 볼 수 있습니다. D-day/Goal로 탭이 나뉘어 있으니 참고 바랍니다.
6. 구글 캘린더 연동 기능(구글 캘린더 api 이용) - 1개 이상의 구글 캘린더 일정을 연동할 수 있습니다. (여러 개 가능)
7. 기본적으로 한국의 공휴일을 구글 캘린더에서 연동해 붉은색으로 표시하게 되어 있습니다. 원치 않을 시 코드 수정이 필요합니다.
8. 갠홈의 메인 페이지(현재는 index.php 및 테이블명이 main으로 설정된 게시판 화면)에 관리자로 로그인했을 때만 볼 수 있는 플로팅 위젯을 띄웁니다.
   플로팅 위젯에 대해: 일정을 추가할 때 위젯 등록 여부에 체크한 일정들을 보여줍니다. 체크하지 않으면 나타나지 않습니다. 코드를 수정해서 모든 방문자가 위젯을 볼 수 있게 하고 보여주고 싶은 일정만 등록하는 것도 괜찮은 방법이에요. (해당 코드는 주석을 달아놓았으니 calendar_widget.php를 참고해 주세요.)
9. 테마 컬러 변경 및 헤더 이미지 지정이 가능합니다. 좌측 상단의 도트들이 테마 컬러 버튼입니다. 대부분 파스텔톤이고, 어두운 갈색 톤의 다크 모드 테마가 하나 준비되어 있습니다.
10. 헤더 이미지의 경우 플레이스홀더를 클릭하거나 좌측 상단 카메라 버튼을 이용해 주세요. 외부링크와 파일 첨부(5MB 미만)가 가능합니다. 외부링크는 저장 시 서버가 이미지를 내려받아 data/calendar_header에 보관하므로 새로고침/캐시 삭제 이후에도 유지됩니다. 기존 이미지를 교체하거나 제거하면 이전 파일은 함께 정리됩니다.

- 배포 예정이 없어서 구글 캘린더 연동 등의 설명은 별도로 하지 않습니다. 연동 후 동기화 버튼을 누르면 구글 캘린더와 갠홈의 캘린더에 등록되어 있는 일정이 서로 동기화됩니다.
- 단, 여기서 주의할 점: 테스트해보니 같은 날 같은 일정이 있으나 서로 시간이 다를 경우(구글 캘린더에 "세션일정" 오후 1시, 갠홈 캘린더에 "세션일정" 오후 3시) 구글 캘린더의 시간이 우선해서 덮어씌워지는 것 같습니다.(혹은 더 이른 시간 쪽이) 이 부분은 테스트를 좀더 해봐야겠지만 참고해 주세요.
